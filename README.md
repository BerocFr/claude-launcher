# Claude Launcher

Installer macOS clé en main pour Claude, Claude Code et MCP.

## Démarrage rapide

```bash
npm install
npm run dev
```

## Build distributable (.dmg)

```bash
npm run dist
```

## Stack
- Electron 28 + React 18 + TypeScript
- Tailwind CSS (dark theme)
- xterm.js + node-pty (terminal embarqué)
- electron-vite (build system)
